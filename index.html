WHAT IS A DATAMART?

A data mart is a repository of data that is designed to serve a particular group of users. Data marts consolidate information from various data sources for use in single departments or subjects areas.

ACCESS  HOW IT WORKS?

EXTRACT

Data is extracted from operational databases. This can be done in near realtime via an API or in batch jobs via an Extract, Transform and Loading (ETL) tool. 

NEXT STEP

TRANSFORM

Data is then transformed to meet minimum regional data standards using an Extract, Transform and Load tool. 

NEXT STEP

LOAD

Data is then loaded into a centralised datamart. From here it can be used to feed external systems. These systems can either have analytical or operational use cases.
